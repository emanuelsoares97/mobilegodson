VERSÃO MODULAR

Abrir o jogo:
1. Extrair o ZIP.
2. Abrir index.html.

O que ficou modular:

1. Denzel / asas
   - js/config/denzel-visual-config.js
   - Para asas normais do mapa/luta.
   - Para posição geral do Denzel em batalha.

2. Mundo / passagens de mapa
   - js/config/world-config.js
   - Para ajustar sensibilidade das saídas automáticas.

3. Lurei em formas separadas
   - js/config/characters/lurei-normal.js
   - js/config/characters/lurei-guerreiro.js
   - js/config/characters/lurei-dominio.js

4. Cenas separadas
   - js/config/scenes/kraidus-transformation-scene.js
   - js/config/scenes/nilzin-transformation-scene.js
   - js/config/scenes/lurei-shadow-entrance-scene.js
   - js/config/scenes/nilzin-abduction-scene.js
   - js/config/scenes/lurei-reveal-scene.js
   - js/config/scenes/lurei-purification-scene.js

Guia principal:
- docs/MODULAR_CENAS_PERSONAGENS.md
- docs/ONDE_MEXER.md

Nota importante:
Se quiseres mexer na cutscene do Kraidus, usa:
js/config/scenes/kraidus-transformation-scene.js

Se quiseres mexer nas asas normais do jogo, usa:
js/config/denzel-visual-config.js
